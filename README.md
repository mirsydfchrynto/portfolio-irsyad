# [PORTFOLIO] — Irsyad Architect
### Professional Showcase & Engineering Lab

Repository ini berisi source code untuk portfolio pribadi saya yang dibangun menggunakan **Next.js 16** dan di-deploy melalui **Surge**. Proyek ini dirancang untuk menampilkan kapabilitas engineering saya di bidang **Mobile Development** dan **Software Architecture**.

---

### 🚀 TECH STACK (PROJECT ONLY)
*   **Framework:** Next.js 16 (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **Deployment:** Surge (Automated via CLI)
*   **Features:** Executive Grid UI, Automated CV Generator (PDF), Base64 Image Delivery.

---

### 📂 PROJECT ARCHITECTURE
Portfolio ini mengikuti struktur modern Next.js:
*   `src/app`: Definisi rute dan layout utama.
*   `src/components`: Komponen UI modular (CodeWorkspace, CommandPalette, dll).
*   `src/config`: Data statis untuk portfolio dan konfigurasi generator PDF.
*   `public`: Aset gambar dan dokumen pendukung.

---

### 🛠️ CARA MENJALANKAN LOKAL
1. Clone repositori:
   ```bash
   git clone https://github.com/mirsydfchrynto/portfolio-irsyad.git
   ```
2. Install dependensi:
   ```bash
   npm install
   ```
3. Jalankan development server:
   ```bash
   npm run dev
   ```

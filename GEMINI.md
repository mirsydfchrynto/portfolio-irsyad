# PORTFOLIO ARCHITECT: DESIGN MANDATES

This document defines the architectural and design standards for the `portfolio-irsyad` project. It focuses on a professional, high-integrity technical aesthetic.

---

## 1. DESIGN PHILOSOPHY
The visual language is defined by the intersection of **Clean Minimalism**, **High-Contrast Brutalism**, and **Technical Integrity**.

### 1.1 Color Palette
- **Main:** #060608 (Deep Black) and #FFFFFF (White).
- **Accent:** #E31B23 (Signature Red). Used for focus and identity.
- **Logic:** Colors should be used purposefully to highlight engineering details and identity.

### 1.2 Typography
- **Display:** Space Grotesk. Used for headers.
- **Sans:** Inter / SF Pro Display. Used for reading.
- **Mono:** JetBrains Mono. Used for technical context and logs.

### 1.3 Components
- **Brutal Cards:** Sharp corners, subtle borders (white/5 or white/10), and deep shadows.
- **Grid Background:** Subtle 80px/20px grid to provide structure.

---

## 2. TECHNICAL STANDARDS
- **Stack:** Next.js 16 (App Router), Tailwind CSS 4, Framer Motion.
- **Structure:** Modular components but keeping the main landing page flow intuitive.
- **Responsiveness:** Priority on ensuring layout integrity across all devices (Mobile to Ultra-wide).

---

## 3. CONTENT STRATEGY
- **Tone:** Professional, direct, and grounded in technical reality.
- **Focus:** Showcasing real-world constraints, technical reflections, and engineering journeys.
- **Projects:** Highlight "Secure CBT Master", "Geges Smart Barber", and "VisionSafe".
